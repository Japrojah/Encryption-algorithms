This is a little Python program that I did as a term paper at the university. The program encrypts data based on RSA, EGSA, DSA, GOST 34.10-94 algorithms.
An encrypted message can be sent to another person. In order to read the message, he will need to do the reverse, knowing only your key. 
My initial data: RSA - [m=15; p=217; q=213; e=5]
EGSA - [p=151; g=147; x=39; k=77]
DSA - [G=111; P=113; q=56; x=23; k=50]
GOST 34.10-94 [P=113; q=56; a(g)=114; x=22; k=50]
